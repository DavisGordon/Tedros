Sistemas corporativos customizados

Através do nosso sistema Tedros é possível atender todas as necessidades de que seu negocio precisa.

Tedros é um sistema gerenciador de aplicativos e podemos adicionar, remover ou alterar aplicativos de que seu negocio precisa, por padrão o Tedros vem com os seguintes aplicativos que podem ser customizados para atender sua necessidade:

  • Ferramentas, possui o módulo Themas para customizar o seu Tedros com as cores e plano de fundo da sua marca.
    Controle de Autorização, crie e gerencie os perfis de acesso aos aplicativos e os seus devidos usuários.
    Preferências, com as definições do sistema como, por exemplo, a definição do logotipo que deve aparecer nos relatórios e ate mesmo a configuração do servidor SMTP para o envio de e-mail pelo módulo de Notificação onde é possível agendar o envio de e-mails.
  • Suporte, neste aplicativo encontra-se alguns dos módulos básicos para integração com a maioria das soluções de que seu negocio vai precisar.
    Artefatos Documentais para gerar ou guardar documentos (arquivos). Integrado ao módulo de notificação é possível enviar o documento por e-mail.
    Localização, este item possui módulos para editar informações de Países, Estados, Cidades, Endereços e Lugares. Podendo estes dados importados através de um arquivo xls, xlsx ou csv. A solução já vem com estes arquivos para importação com uma grande base de dados e a edição de endereços pode ser acompanhada e visualizada através de um mapa.
  • Pessoas e Empresas aqui se encontram os módulos para gerenciar Clientes, Pessoas Físicas e Pessoas Jurídicas podendo gerar relatórios dos mesmos. Fornecimento de serviços, este aplicativo possui módulos para gerenciar os serviços, contratos e planos.

Além destes aplicativos base o sistema vem com a funcionalidade de Chat, recurso de envio de mensagens entre os usuários do sistema.

Ao realizarmos uma analise na sua necessidade identificaremos os aplicativos e módulos que deverão ser criados para atender seu negocio e integra-los aos aplicativos base.

O sistema Tedros foi desenhado para funcionar sob uma topologia cliente-servidor, isto é, enquanto você usa o Tedros (cliente) no seu computador o processamento das informações é realizado em outro (servidor).

Caso sua empresa não tenha uma infraestrutura com servidores disponíveis mas tem acesso a internet podemos fornecer servidores na nuvem (cloud) porem se não tiver acesso a internet os serviços do servidor pode ser instalado no mesmo computador cliente limitando assim a principal característica de atender vários usuários simultâneos.
